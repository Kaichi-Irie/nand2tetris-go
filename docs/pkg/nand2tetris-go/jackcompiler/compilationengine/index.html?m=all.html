func (*CompilationEngine) CompileTerm ¶ 

func (ce *CompilationEngine) CompileTerm(isDoStatement bool) error

CompileTerm compiles a term and writes it to the XML file. Term: integerConstant | stringConstant | keywordConstant | varName | varName '[' expression ']' | subroutineCall | '(' expression ')' | unaryOp term skipTags: if true, do not write the <term> and </term> tags. This is used for subroutine calls.
